# Dental Clinic Management System

A web-based application designed to help dental clinics manage patients, appointments, procedures, communication, and income records efficiently.

---

## Features

- Patient record management
- Appointment scheduling system
- Treatment and procedure tracking
- Income and payment records
- Role-based access control (Admin & Assistant)
- Messaging functionality (currently under development / optional)

---

## User Roles

### Admin
- Full access to all features
- Can add, edit, and delete patient records
- Can view income and financial reports

### Assistant
- Can add patients and procedures
- Restricted from deleting records
- No access to income data

---

## Tech Stack

- Python (Backend)
- Flask / Django (Framework)
- HTML, CSS, JavaScript (Frontend)
- SQLite / Database (if used)
- Hosted on PythonAnywhere

---

## Project Status

This is a working prototype deployed for demonstration purposes. Some features (like messaging automation) are under development.

---

## Purpose

To digitize and simplify dental clinic operations by replacing manual record-keeping systems with an easy-to-use web-based solution.
